# Final Demo Flow

## 1) Show System Scope
- Open system assessment/scope artifacts and define in-scope trust boundaries.

## 2) Show Architecture Diagram
- Present architecture/mapping artifacts and point to control insertion points.

## 3) Show Threat Model
- Walk through threat model and abuse-case assumptions.

## 4) Show Retrieval Authorization Control
- Demonstrate retrieval authorization flow and policy expectation.

## 5) Show RAG Boundary Test
- Show retrieval-boundary test dataset/plan and expected pass criteria.

## 6) Show Audit Event
- Demonstrate policy/audit event expectations and logging evidence path.

## 7) Show Runtime Trace
- Show how runtime decisions are intended to be reconstructed for investigations.

## 8) Show Evidence Pack
- Walk the evidence summary and classification (Verified / Partially Confirmed / Not Proven).

## 9) Show Dashboard
- Display dashboard artifacts and launch posture indicators.

## 10) Show Launch Gate Decision
- Present conditional hold decision and exact unmet criteria.

## 11) Show Residual Risks
- Review residual-risk categories and treatment owners.

## 12) Show Governance Plan
- Close with continuous review cadence and control ownership accountability.

## Demo Guardrails
- Do not claim real client deployment.
- Keep Atlas Advisory Group clearly fictional.
- Explicitly call out Not Proven items.
