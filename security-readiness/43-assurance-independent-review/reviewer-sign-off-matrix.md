# Reviewer Sign-Off Matrix

## Purpose
Define who must sign, what evidence they rely on, and what decision authority each reviewer holds.

## Sign-off Matrix

| Review Stage | Required Reviewer Role | Decision Authority | Minimum Evidence Required | Sign-off Status | Name | Date (YYYY-MM-DD) | Signature |
|---|---|---|---|---|---|---|---|
| Control Effectiveness Review | Control Owner (First Line) | Recommend only | Control implementation + tests + open risks. | Pending |  |  |  |
| Independent Control Review | Second-Line Reviewer | Approve / Block / Conditional | Independent checklist + evidence quality review + unresolved findings list. | Pending |  |  |  |
| Security Governance Review | Security/GRC Lead | Approve / Block | Risk acceptance records + exception approvals + audit traceability. | Pending |  |  |  |
| Launch Gate Review | Launch Authority | Final Go/No-Go | Consolidated assurance package and signed reviewer artifacts. | Pending |  |  |  |

## Enforcement Rule
Do not mark independent review as completed until all required reviewer signatures are present for the applicable stage.
