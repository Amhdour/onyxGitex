# Demo Flow

## 15-minute version
- Slides: 1,2,3,4,9,10
- Timing: 2m,2m,3m,3m,3m,2m
- Repo files: `pitch-deck-pack/04-slide-evidence-chain.md`, `pitch-deck-pack/deck-status.json`
- Skip: deep scenario-level scripts and detailed runtime walkthrough.

## 30-minute version
- Slides: 1 through 10
- Timing: ~3 minutes each
- Repo files: all slide files + `deck-claim-boundaries.md` + `deck-one-page-summary.md`
- Explain deeper: Version 2A/2D scenarios and Version 4 fill requirements.

## 60-minute workshop version
- Slides: 1 through 10 + live file walk
- Exercises: map one client use case to Version 4 evidence fields.
- Questions: identity model, tool risk, data classes, incident response expectations.
- Client inputs to collect: scope, roles, data boundaries, tools, approvals, compliance owners, risk acceptance owners.
