# Visual Style Guide

- Palette: black/white/neutral grays.
- Layout: clean consulting-style with strong spacing and minimal text clutter.
- Icons: thin-line icons only.
- Diagrams: evidence-chain flows, trust boundaries, maturity ladder.
- Avoid: exaggerated cyber imagery, fake dashboards, fake logos.

## Recommended fonts
- Inter
- Source Sans 3
- IBM Plex Sans

## Color guidance
- Primary text: #111111
- Secondary text: #444444
- Divider gray: #D9D9D9
- Accent neutral blue-gray: #5B6B7A

## Slide density guidance
- 1 core message per slide
- 3–8 bullets max
- Keep left alignment and consistent margins
