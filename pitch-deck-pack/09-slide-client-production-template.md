# Client-Specific Production Template

## Slide title
Client-Specific Production Template

## Slide purpose
Explain this stage of the readiness story with clear claim boundaries.

## Main message
Production readiness cannot be universal. A real client decision requires real client evidence.

## Visual idea
Use a simple consulting-style diagram with neutral palette and minimal iconography.

## Talk track
Focus on evidence boundaries, launch-gate logic, and what is still blocked without real client evidence.

## Key bullets
- Client scope
- Client data
- Client users and identity
- Client tools and agents
- Runtime evidence
- Compliance and legal review
- Launch approval
- Residual risk acceptance
- Named approver signoff
- Current status: CLIENT_PRODUCTION_TEMPLATE_READY
- Boundary: client evidence not verified; current decision remains NO-GO.

## What not to say
- This is production-ready.
- This is fully secure.
- This is a real production GO.

## Client takeaway
Use evidence to decide readiness; do not treat demo success as production authorization.
