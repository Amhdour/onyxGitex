# Deck Claim Boundaries

## Allowed wording
- readiness methodology
- evidence chain
- local runtime evidence
- CI artifact proof
- observability correlation
- local agent runtime evidence
- staging-demo evidence model
- client-specific production template
- production readiness remains blocked until real client evidence exists

## Forbidden wording and replacements
- production ready → production readiness remains blocked pending real client evidence
- fully secure → risk-reduced with evidence-based controls under defined scope
- guaranteed safe → designed for evidence-based risk reduction, not guarantees
- client approved → template ready for client-specific evidence intake
- compliance approved → compliance review required and not yet verified
- real production GO → current decision remains NO-GO pending client evidence
- universal production safety → client-specific readiness evaluation required
- real external integration proof → external integration proof not claimed in this pack
- real LangGraph production proof → local deterministic agent runtime evidence only
