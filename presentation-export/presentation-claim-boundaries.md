# Presentation Claim Boundaries

## Allowed wording
- readiness methodology
- evidence chain
- local runtime evidence
- CI artifact proof
- observability correlation
- local agent runtime evidence
- staging-demo evidence model
- client-specific production template
- production readiness remains blocked until real client evidence exists

## Forbidden wording with safe replacements
- production ready → production readiness remains blocked pending real client evidence
- fully secure → risk-reduction approach with bounded evidence scope
- guaranteed safe → evidence-based risk reduction, not guarantee language
- client approved → client evidence intake template prepared; approval not claimed
- compliance approved → compliance review required and not yet verified
- real production GO → GO decision remains false pending client evidence
- universal production safety → safety posture is client-specific and evidence-dependent
- real client evidence verified → client evidence remains unverified in this state
- client GO → client GO is not claimed; decision remains pending
