# 07 Client Engagement Workflow

1. Discovery call — Goal: align scope; Input: use case; Deliverable: discovery notes; Decision impact: go/no-go for assessment start.
2. AI system scope — Goal: define boundaries; Input: architecture summary; Deliverable: scope doc; Decision impact: test coverage range.
3. Inventory — Goal: list assets; Input: systems/tools/users; Deliverable: inventory; Decision impact: control coverage.
4. Data classification — Goal: rank data sensitivity; Input: datasets; Deliverable: classification matrix; Decision impact: control strictness.
5. Architecture mapping — Goal: map trust paths; Input: diagrams; Deliverable: system map; Decision impact: threat surface clarity.
6. Threat modeling — Goal: identify abuse paths; Input: scenarios; Deliverable: threat model; Decision impact: priority controls.
7. Control mapping — Goal: map safeguards; Input: policies; Deliverable: control map; Decision impact: gap visibility.
8. RAG boundary tests — Goal: test retrieval rules; Input: test cases; Deliverable: RAG evidence; Decision impact: retrieval confidence.
9. Agent tool authorization tests — Goal: validate tool governance; Input: tool list; Deliverable: agent evidence; Decision impact: actuation risk.
10. Observability and audit review — Goal: verify traceability; Input: telemetry setup; Deliverable: observability evidence; Decision impact: forensic readiness.
11. Incident-readiness review — Goal: verify response model; Input: runbooks; Deliverable: incident readiness notes; Decision impact: recovery confidence.
12. Evidence pack generation — Goal: consolidate proof; Input: all artifacts; Deliverable: evidence pack; Decision impact: gate decision basis.
13. Residual risk review — Goal: label unresolved risk; Input: exceptions; Deliverable: risk register; Decision impact: acceptance conditions.
14. Launch-gate decision — Goal: NO-GO/CONDITIONAL-GO/GO; Input: evidence + approvals; Deliverable: decision record; Decision impact: launch posture.
15. Handoff and continuous governance — Goal: sustain controls; Input: owners; Deliverable: operating model; Decision impact: long-term readiness.
