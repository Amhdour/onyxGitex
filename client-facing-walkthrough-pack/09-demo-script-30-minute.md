# 09 Demo Script (30 Minutes)

Each segment includes what to show, what to say, what not to say, and likely questions.

- **0:00–3:00 — Context and problem**
- **3:00–7:00 — Project structure**
- **7:00–12:00 — RAG readiness evidence**
- **12:00–16:00 — CI and artifact evidence**
- **16:00–20:00 — Observability evidence**
- **20:00–24:00 — Agent runtime evidence**
- **24:00–27:00 — Staging-demo evidence**
- **27:00–30:00 — Client-specific production template and next steps**

Do not claim production approval or universal safety; position this as methodology plus evidence discipline.
