# 17 How to Use Version 4 Template

1) Fill client scope. 2) Fill system inventory. 3) Fill data classification. 4) Fill user identity model. 5) Fill tool and agent inventory. 6) Fill RAG boundary rules. 7) Fill policy and approval rules. 8) Fill compliance constraints. 9) Fill logging and audit requirements. 10) Fill incident-response workflow. 11) Attach runtime evidence. 12) Attach observability evidence. 13) Attach red-team/eval results. 14) Record residual risks. 15) Record approver signoff. 16) Make NO-GO / CONDITIONAL-GO / GO decision.

**Warning:** Until these fields are filled with real client evidence, the decision remains NO-GO.
